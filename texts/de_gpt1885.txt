Das Programm prüft zunächst, ob alle verfügbaren Ressourcen des Arbeitsspeichers für den Start von Nokia-Apps benötigt werden. Dies kann sich zum Beispiel auf Dienste beziehen, die von anderen Herstellern zur Verfügung gestellt wurden. Nach Abschluss der Überprüfung wird ein Server unter Windows 7 oder Windows 8 ausgeführt. Auf dem Server kann optional eine Testumgebung eingerichtet werden.

Außerdem können Sie ein eigenes Icon vor die neue Spalte setzen. Über diesen Eintrag wird dann das Event automatisch beim Start des Windows-Dienstes ausgeführt. Auf dem Reiter System können Sie Windows-Funktionen konfigurieren. In der rechten Spalte finden Sie ein Drop-Down-Menü, das alle Dienste des Windows Server bietet. Wenn bei Ihnen im Browser ein neuer Tab geöffnet wird, wird die Windows-Funktion standardmäßig in der Seitenleiste des Browsers angezeigt. Sie können das Aussehen durch ein CSS-Stylesheet verändern.

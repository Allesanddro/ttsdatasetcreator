Sie trug eine Jacke, die so aussah wie eine große Tasche voller Geld. Es war eine Lederjacke, so dick, so viele Falten, so viel Plunder, dass ich mich fragte, wie man so etwas tragen konnte. Und sie war so wunderschön. So viele feine kleine Falten. Und diese großen, schwarzen Augen. Ich sah, wie sie auf den Schultern ihrer großen Schuhe ging.

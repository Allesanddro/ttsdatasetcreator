Die meisten Browser geben diese Informationen durch den URL-Parameter für die Dokumentvorlage an. So wird das Einbetten nur dann von dem neuen Dokument akzeptiert, wenn der Name des Objekts, für das dieser Parameter an den Browser gesendet wird, auf einer Website als Überschrift in Ihrer Dokumentvorlage verwendet wird. Wählen Sie im Dokumentfenster einen Ordner für das PDF-Dokument aus. Wenn Sie Ihr Dokument oder PDF zum Veröffentlichen vorbereitet haben, können Sie den Ordner auswählen, den Sie als Standardspeicherort für die PDFs festlegen wollen.
